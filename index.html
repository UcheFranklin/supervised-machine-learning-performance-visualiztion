Introduction

Web Interface for the Performance Visualization of Supervised Learning Algorithms

This is web-based interface designed and developed to present the results of the performance of the seven supervised machine learning algorithms for the prediction of the mortality chance of hepatitis and related ailments as well as the comparison between each of them. The data and evaluation in this web interface are obtained from WEKA testing results and the predictive model generated. Simple graphical representation are used to show the results of the performance and comparison in this web-based interface.
